Workflow Blocks

Cube Modelling

Project:

When I started getting interested in data visualisation, I wanted something physical to experiment with. Lego was an option but not only the cost was something to consider but also the variety of brick options. I found this cube block set which I thought was ideal with one size and a limited but varied colour palette.

Process:

This was one of the first experiments I used with cube blocks as a prototype to visualise a project workflow model.
